# AI Data Assistant

A Python tool that uses OpenAI + Pandas + MySQL to analyze data from CSV, Excel, or database.

## Run

1. Create `.env` file:

OPENAI_API_KEY=your_key
DB_URL=mysql+pymysql://ai_user:password@localhost:3306/ai_project

2. Install requirements:

pip install -r requirements.txt

3. Run:

python csv_ai_assistant.py
